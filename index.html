fRest: REST with Flask

A POC python implementation to show GET/POST/PUT/DELETE REST APIs using Flask library

Installation

  • pip install flask
  • python service.py

Usage:

GET Console:

http://localhost:8001/

Get All Users:

http://localhost:8001/users/all

ADD an User

http://localhost:8001/users/userinfo

Update an User

http://localhost:8001/users/user/update/id/

Delete an User

http://localhost:8001/users/user/remove/id/
